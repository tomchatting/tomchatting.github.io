accessibility statement for tomwillgoto.space

This statement covers the tomwillgoto.space website. It does not cover any subdomains.

This website is run by Thomas Chatting.

How you should be able to use this website

I want as many people as possible to be able to use this website. For example, that means you should be able to:

  • change colours, contrast levels and fonts using browser or device settings
  • zoom in up to 400% without the text spilling off the screen
  • navigate most of the website using a keyboard or speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)
  • I try to make the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

Technical information about this website's accessibility

I have used the IBM Equal Access Accessibility Checker to test my website against it's compliance to the Web Content Accessbility Guidelines (WCAG) 2.2. This tool confirms my website's compliance to this standard.

If you have any feedback regarding the accessibility of my website, or it's content, please contact me. The details to contact me are contained in the footer section of every page on this site.
